# Product Overview

This is an economics research project that combines data engineering, natural language processing (NLP), and machine learning to analyze economic data. The project works with structured datasets (CBS dataset in RDS format) alongside research papers (PDFs), applying ML and NLP techniques — including transformer-based models with LoRA fine-tuning — to extract insights, classify text, or model economic phenomena.

Key capabilities:
- PDF and web scraping for data collection
- NLP pipelines using spaCy and Hugging Face transformers
- Deep learning with TensorFlow and PyTorch
- Statistical modeling with statsmodels and scikit-learn
- Experiment tracking via MLflow and Weights & Biases
- Data versioning with DVC
